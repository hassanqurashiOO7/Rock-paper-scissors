# Rock Paper Scissors 🎮

A simple command-line Rock Paper Scissors game built with Python.

## How to Play

1. Clone the repo and navigate to the folder
2. Run the game:
   ```
   python rock_paper_scissors.py
   ```
3. Enter your choice each round:
   - `r` → Rock
   - `p` → Paper
   - `s` → Scissor
   - `n` → Quit the game

## Features

- Play unlimited rounds against the computer
- Score is tracked for both you and the computer
- Final score is shown when you quit

## Requirements

- Python 3.x (no external libraries needed)

## Example

```
=== Rock Paper Scissors ===
Enter r (rock), p (paper), s (scissor), or n to quit

Enter your choice: r
You win! Computer: scissor | You: rock
Score → You: 1 | Computer: 0

Enter your choice: n

Game Over! Final Score → You: 1 | Computer: 0
```
